# Описание проекта

Главная цель проекта - создание обучающего веб-сервиса по веб-разработке.

Планируется разместить структурированные материалы курса: теоретические разделы, практические задания, примеры кода и дополнительные рекомендации для закрепления знаний.
После изучения отдельных тем пользователи смогут проходить тесты, проверять уровень понимания материала и видеть свои результаты. Это поможет.

Приложение разделим на две части 
    - *Информационный портал с материалами* 
    - *Система тестирования*
