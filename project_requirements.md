# Что должно быть реализовано в проекте

В проекте необходимо реализовать обучающий веб-сервис, посвященный одному курсу по веб-разработке. На сайте пользователи смогут изучать материалы этого курса, проходить тесты и отслеживать свой прогресс.

## Основные разделы сайта

На сайте должны быть реализованы основные страницы:

- главная страница с кратким описанием проекта;
- страница со списком разделов одного курса;
- страница отдельного учебного материала;
- страница с тестами по темам курса;
- страница результатов тестирования;
- страница регистрации и входа пользователя;
- личный кабинет пользователя.

## Учебные материалы

Необходимо реализовать возможность размещения материалов одного курса по веб-разработке. Материалы должны быть разделены по темам и иметь понятную структуру.

В учебных материалах могут быть:

- текстовые объяснения;
- примеры HTML, CSS, JavaScript и Python-кода;
- изображения и схемы;
- практические задания;
- ссылки на дополнительные источники.

## Система тестирования

В проекте должна быть реализована система тестов для проверки знаний пользователей.

Тесты должны включать:

- вопросы по темам курса;
- несколько вариантов ответа;
- проверку правильности ответов;
- подсчет итогового результата;
- отображение количества правильных и неправильных ответов;
- возможность повторного прохождения теста.

## Пользователи

В проекте необходимо реализовать регистрацию и авторизацию пользователей. После входа пользователь должен иметь доступ к личному кабинету.

В личном кабинете должно отображаться:

- имя или логин пользователя;
- список пройденных тестов;
- результаты тестирования;
- общий прогресс обучения.

## Административная часть

Для управления проектом должна быть предусмотрена административная часть.

Администратор должен иметь возможность:

- добавлять и редактировать учебные материалы;
- создавать и изменять тесты;
- добавлять вопросы и варианты ответов;
- просматривать пользователей;
- просматривать результаты тестирования.

## Дизайн и удобство использования

Интерфейс сайта должен быть простым, понятным и удобным. Пользователь должен быстро находить нужные материалы и легко переходить между разделами.

Важно реализовать:

- адаптивную верстку для компьютеров и мобильных устройств;
- удобное меню навигации;
- понятное оформление учебных материалов;
- визуальное выделение результатов тестов;
- единый стиль всех страниц сайта.
