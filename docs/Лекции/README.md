# Подготовленные лекционные материалы

Материалы подготовлены на основе транскрибированных записи занятий.

## Состав

- `00_введение_в_курс_и_история_веба.md` — организация курса, требования к проекту, инструменты, история веба, открытые системы и API.
- `01_http_клиент_сервер_и_архитектура_веба.md` — клиент-серверная архитектура, HTTP, URL, методы, заголовки, статусы, HTTPS и nginx.
- `02_html_css_django_первое_практическое_занятие.md` — HTML, CSS, Bootstrap, `uv`, создание Django-проекта, миграции, админка, модели и первые view.

## Формат каждого материала

- Аннотация.
- Цели занятия.
- План.
- Структурированная теория.
- Примеры кода.
- Термины.
- Контрольные вопросы.
- Практическое или домашнее задание.
