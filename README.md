# Учебные материалы по курсу Django Vue Lab

Этот репозиторий содержит учебные материалы по курсу `Django` и связанным темам

## Содержание

- `docs/` - заметки, черновики и материалы курса
- `docs/Презентации/` - готовые презентации и исходники в `Typst`
- `docs/Excalidraw/` - схемы и наброски

## Презентации

В папке `docs/Презентации/src/` лежат исходники слайдов:

- `lect00_intro.typ` - введение в курс
- `lect01.typ` - `HTTP`, `HTML`, `CSS`

Собранные версии лежат рядом в виде `PDF`.
